# BigDesign

## Project setup
1. Open `project/BigDesign.2008.xpr` with Vivado (tested with 2025.2).
1. Setup `my_project.vhdl`:
   * Rename `src/PoC/my_project.vhdl.template` to `src/PoC/my_project.vhdl`.
   * Fill in the fields according to the project directory and operating system.

## Run simulation
1. Open your preferred simulator (tested with Aldec Riviera-PRO 2025.10).
1. Navigate to the `temp` folder.
1. Run the `regression.tcl` file.
